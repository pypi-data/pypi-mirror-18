ardj: internet radio software

ARDJ is a software to operate an internet radio in automatic mode. It integrates with ices/ezstream, communicates with admins and listeners using the XMPP protocol (Jabber), can send messages to twitter, download music from Last.fm and Jamendo, and more.

Distributed under the GNU GPL.

System architecture

ARDJ based radio consists of 3 main working components:

  • The streaming server. It gives out the MP3/Ogg stream to your listeners. This is usually icecast.
  • The encoder. It reads the playlist, chooses tracks to play, encodes the MP3/Ogg stream and sends it to the streaming server. This is usually ezsteram or ices. Usually these programs support plugins which pick files to play, that's where ardj comes in. Using a database and a sophisticated randomizer it tells ezstream which file to play next.
  • The web based control center. You can use it to upload new music, schedule tracks ahead of the playlist, etc.
