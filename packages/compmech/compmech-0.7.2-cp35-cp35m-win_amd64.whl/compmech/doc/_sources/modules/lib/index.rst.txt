
Compmech shared libraries (lib)
===============================

This is not a Python importable module, but instead it contains all shared
libraries, programmed in C, that are used among various codes.

The following libraries are currently available:


- :ref:`Bardell's approximation functions <theory_func_bardell>`
- :ref:`Gaus-Legendre quadrature rule <theory_integrate_legendre_gauss>`
